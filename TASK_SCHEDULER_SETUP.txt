# Windows Task Scheduler Setup for Job Hunter Emails

## Manual Setup Instructions:

1. Open Windows Task Scheduler:
   - Press Win + R, type `taskschd.msc`, press Enter

2. Create New Task:
   - Click "Create Task..." in the right panel
   - Name: `JobHunterEmailSender`
   - Description: `Sends job application emails automatically at 8 AM`

3. Triggers Tab:
   - Click "New..."
   - Begin the task: `On a schedule`
   - Settings: `Daily`
   - Start: `8:00:00 AM` (or your preferred time)
   - Click OK

4. Actions Tab:
   - Click "New..."
   - Action: `Start a program`
   - Program/script: `C:\Users\saite\OneDrive\Documents\Agent applies\run_emails.bat`
   - Click OK

5. Conditions Tab:
   - Check "Start the task only if the computer is on AC power" (uncheck if you want it to run on battery)
   - Check "Start only if the following network connection is available: Any connection"

6. Settings Tab:
   - Check "Run task as soon as possible after a scheduled start is missed"
   - Check "If the task fails, restart every: 1 minute, up to 3 times"
   - Check "Stop the task if it runs longer than: 1 hour"
   - Check "If the running task does not end when requested, force it to stop"

7. Click OK and enter your password when prompted

## How it works:
- Task runs daily at 8 AM
- Executes the batch file which runs the Python script
- Automatically answers "yes" to send emails
- Logs activity to email_scheduler.log
- Laptop can be closed - Windows will wake it if needed

## To test immediately:
Run: `run_emails.bat`

## To modify schedule:
- Open Task Scheduler
- Find "JobHunterEmailSender" task
- Right-click → Properties → Triggers tab
